# WebExtension Module

## Purpose

The `webExtension` module covers installation and removal of browser extensions.

## Why It Matters In `ruyiPage`

It provides the protocol-side framing for extension-oriented browser-control workflows.

## Core Standard Areas

- `webExtension.install`
- `webExtension.uninstall`

## Practical `ruyiPage` Alignment

Most relevant local examples:

- `22_web_extension.py`

## AI Guidance

- treat this as a specialized capability, not a default path for ordinary page automation
- mention it only when extension setup is part of the actual task
